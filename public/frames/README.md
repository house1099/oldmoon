# frames

頭像框與卡片外框 PNG 資產目錄。

## 目錄

- **`frames/` 根目錄**：legacy；舊路徑仍可放在此處。
- **`avatars/`**：頭像框（`item_type === avatar_frame`）建議放這裡。
- **`cards/`**：卡片外框（`item_type === card_frame`）建議放這裡。

## 頭像框（avatars）

- 建議尺寸：`512×512` px
- 格式：`PNG`
- 背景：透明
- 路徑範例：`/frames/avatars/star-frame.png`（或 legacy：`/frames/star-frame.png`）
